# jwt_flask_authentication

## JWT(Jason Web Tokens)

+ JSON Web Tokens is a secure and compact way to transmit data between two parties with the help of JSON objects.

+JSON web token consists of three parts, namely

    * Payload
    * Header
    * Signature
    
## Steps to implement

```
Flask-RESTful==0.3.8
```
